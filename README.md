# Lucas Martin — Portfólio

Portfólio pessoal de Lucas Martin, com foco em desenvolvimento de sistemas, automação de processos, redes, segurança aplicada e projetos digitais com uso real.

Esta versão foi criada como uma vitrine minimalista: menos dashboard, mais apresentação. O objetivo é mostrar quem sou, o que venho construindo e quais ideias fazem parte da minha fase atual.

## Destaques

- **Sentinela IF / Cameras-IF**: videomonitoramento institucional com WebRTC, Tailscale, Docker e acesso privado.
- **Deliver-Bot**: automação operacional para WhatsApp, processamento de PDFs, gestão de filiais e painel web.
- **Dieta Ideal Cabras**: app Flutter premiado com 1º lugar no nível médio construtivo da XI FECIB para apoio ao manejo nutricional de cabras leiteiras.
